# JLU Smart Car - 吉林大学智能车实验项目 🚗

本项目是吉林大学智能车实验课程的完整代码实现。基于 STM32 微控制器开发，采用模块化状态机架构，实现了蓝牙无缝切换的三种核心控制模式：红外加权循迹、陀螺仪闭环直行、以及 PID 速度闭环遥控。

## ✨ 核心特性

- **多模式状态机无缝切换**：支持运行时强行切断/重置状态，杜绝模式“窜台”。
- **硬件级防死锁**：开机底层重排时序，解决蓝牙连机瞬间导致串口 ORE 溢出中断锁死的问题。
- **动态控制流路由**：针对不同任务自动路由底层算法（PID 闭环 vs 纯 PWM 直接驱动）。
- **抗漂移陀螺仪算法**：启用 MPU6050 硬件 DLPF 低通滤波，结合微小死区滤除零飘。

## 🛠️ 硬件清单

- **主控芯片**: STM32 系列 (利用 TIM 定时器及中断管理)
- **姿态传感器**: MPU6050 (通过 I2C 读取 Z 轴偏航角 Yaw)
- **循迹模块**: 4 路红外光电传感器 (车头对称分布：距离中心 1.5cm 与 3cm)
- **通信模块**: JDY-31 (或其他串口蓝牙透传模块)
- **驱动执行**: 带有霍尔/光电编码器的直流减速电机 + 电机驱动板

## 🎮 工作模式说明

小车上电后，蓝牙会自动握手连接，系统默认进入 **Mode 0 (安全等待模式)**，此时小车锁定刹车，等待用户指令。通过手机蓝牙串口助手（文本/ASCII 模式）发送对应数字即可切换模式：

### `1` -> Mode 1: 极速红外循迹模式 (Line Tracking)
- **逻辑**: 纯 PWM 驱动，不经过 PID 和陀螺仪。
- **算法**: 采用基于物理距离真实比例的 **1:5 加权误差算法**。内侧传感器压线微调，外侧传感器压线提供 5 倍权重的差速强力回正。
- **保护**: 具备“肌肉记忆”功能，在十字路口全黑或全白冲出赛道时，能够继承上一刻的转向趋势找回黑线。

### `2` -> Mode 2: 陀螺仪锁尾直行模式 (Gyro Straight)
- **逻辑**: 速度 PID 内环 + 陀螺仪偏航角外环（双闭环）。
- **算法**: 记录开机时的 Yaw 角度作为 Target，行进中不断对偏航角误差进行比例补偿（Turn_Compensate），转化为左右轮的一正一负差速，使小车像拥有“锁尾陀螺”一样笔直前进。
- **保护**: 带有物理限幅锁，防止遭受外部剧烈撞击时补偿值过大导致单侧轮挂倒挡甩尾。

### `3` -> Mode 3: 蓝牙键盘遥控模式 (RC Mode)
- **逻辑**: 单纯的速度 PID 闭环控制。
- **操作**: 发送模式指令 `3` 后，小车静止。随后可通过蓝牙发送方向字符控制：
  - `F` (Forward): 前进
  - `B` (Backward): 后退
  - `L` (Left): 原地逆时针左自转
  - `R` (Right): 原地顺时针右自转
  - `S` (Stop): 紧急刹车
